# 💼 Portfólio Corporativo — Geovani de Matos Gonçalves

Este é um projeto de portfólio profissional e comercial desenvolvido sob medida para destacar competências executivas em **Liderança de TI, Governança Corporativa, Segurança da Informação (Red Team/Defesa Cibernética) e Compliance Regulatório (LGPD)**.

O objetivo principal foi transformar uma estrutura estática em uma plataforma premium, minimalista e de alta performance para apresentar marcos de carreira, imersões técnicas e artigos de liderança de forma imponente.

---

## 🚀 Tecnologias Utilizadas

A arquitetura do projeto foca em performance extrema, SEO limpo e responsividade fluida para o mercado corporativo:

*   **HTML5 Semântico & CSS3** — Estruturação sólida e otimizada para navegadores.
*   **Tailwind CSS** — Framework utilitário para design responsivo e gerenciamento nativo de temas (*Dark/Light Mode*).
*   **Alpine.js** — Framework JavaScript minimalista para controle de estados dinâmicos (como o menu mobile e a lógica do carrossel).
*   **Google Fonts** — Tipografia integrada utilizando as famílias *PT Sans* (para títulos de impacto) e *DM Sans* (para máxima legibilidade).

---

## ✨ Funcionalidades Implementadas

*   **Design Minimalista High-Tech** — Interface sofisticada utilizando tons de cinza escuro corporativo com contrastes em verde limão vibrante (`#6de627`).
*   **Menu Computador Otimizado** — Cabeçalho alinhado à esquerda com o novo logotipo unificado (`Geovani Matos`) e navegação fluida em sequência.
*   **Carrossel de Eventos Responsivo** — Seção de "Eventos & Imersões Tech" controlada puramente via Alpine.js, adaptando-se automaticamente para exibir 1 card no mobile e 3 cards no desktop.
*   **Módulo de Blog Pronto para Produção** — Estruturação de páginas separadas (`blog.html` e `blog-article.html`) incluindo uma barra de progresso de leitura em tempo real no topo e artigos técnicos direcionados.
*   **Acessibilidade e Performance** — Layout leve com transições suaves e contraste adaptado para leitura confortável em qualquer dispositivo.
